# 🚀 CRUD Angular + Spring Boot

Aplicação fullstack para operações CRUD (Create, Read, Update, Delete) utilizando:

- 🔹 Frontend: Angular
- 🔹 Backend: Spring Boot
- 🔹 API REST para comunicação entre frontend e backend

---

# Como rodar
Frontend (Aplicação Angular)
-> crud-angular-spring\crud-angular> npm start
Backend (Spring Boot)
-> crud-angular-spring\crud-spring> mvn spring-boot:run


# 📌 📖 Sobre o projeto

Este projeto demonstra a construção de uma aplicação fullstack moderna utilizando Angular no frontend e Spring Boot no backend.

A aplicação permite realizar operações básicas de CRUD, servindo como base para sistemas mais complexos.

---

# 🧱 🏗️ Arquitetura


Frontend (Angular) -> HTTP (REST API) Backend (Spring Boot) -> Database
